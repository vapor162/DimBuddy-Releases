# Dim Buddy Releases

Official installation packages and update metadata for the modern Dim Buddy
SOLIDWORKS add-in.

This repository intentionally contains **release files only**. The Dim Buddy
source code and the legacy VB.NET project are maintained separately.

## Current beta

- Version: `0.9.0-beta.1`
- SOLIDWORKS: 2024 or later
- Windows: 64-bit
- Runtime: .NET Framework 4.8

Download the current package from the `downloads` folder. Close SOLIDWORKS,
extract the ZIP, and run `Install-DimBuddy.cmd` as administrator.

The installer preserves macro assignments, custom icons, themes, favorites,
section states, and SOLIDWORKS toolbar customization during upgrades.
